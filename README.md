# favourite-places

## Цель работы

Cоздание мобильного приложения для хранения большого объема информации о посещенных местах, позволяющее:

– просматривать информацию о сохраненных местах,

– добавлять, редактировать, удалять информацию о месте,

– искать в списке место по наименованию или местоположению,

– сортировать список мест по возрастанию или убыванию по следующим параметрам: дата добавления в приложение, наименование, рейтинг,

– просматривать на географической карте метки, соответствующей адресу или координатам места,

– получать адрес места при помощи обозначения метки на географической карте,

– построить автомобильный или пешеходный маршрут от текущего местоположения пользователя к выбранному месту.

Note: Актуальная версия приложения находится на ветке develop
